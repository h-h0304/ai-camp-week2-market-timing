## 1. 빠진 정보 — 점수: 6/10 (문제 있음)

골든에는 있는데 초안에 빠진 항목들.

### 문제점

**① 헤더에 `검색 기간` 필드 누락**
- 골든: `**검색 기간**: 2025.05.22 ~ 2025.06.22 (작년 동기 1개월)`
- 초안: 해당 줄 없음. 대신 `분석 대상` 필드가 있음
- 왜 문제: 검색 기간은 데이터 출처의 범위를 명시하는 핵심 정보. 없으면 데이터 신뢰성 판단 불가.

**② 시즌 분석에 `남은 판매 기간` 없음**
- 골든 2위(흑수박, 시즌 초입): "남은 판매 기간: 약 3개월"로 현재~시즌 끝까지 기간 명시
- 초안: 모든 품목에 "남은 준비 기간"만 있고 남은 판매 기간 없음
- 왜 문제: 시즌 전체 중 어느 위치인지 파악 불가. 진입 후 얼마나 팔 수 있는지 알 수 없음.

**③ 최종 주의사항에 배송 인프라 문구 누락**
- 골든: `배송 인프라 (냉장/상온) 사전 준비 필수`
- 초안: 해당 항목 없음
- 왜 문제: 실제 운영에서 배송 인프라 미준비는 치명적 리스크. 공급처 확보와 별개로 반드시 체크 필요.

**④ 탈락 품목 섹션 없음**
- 골든: "청송사과, 대봉감: 시즌 종료 → 진입 불가"처럼 분석했지만 탈락한 품목을 명시
- 초안: 3개 모두 진입 가능이라 탈락 품목 자체가 없어 해당 사항 없음 — 단, 형식 구조로 "탈락 품목 없음" 한 줄이라도 남겨야 골든 형식에 완전히 부합

## 2. 구조 차이 — 점수: 7/10 (명확한 이탈)

섹션 순서·형식이 골든과 다른 부분.

### 문제점

**① 시즌 분석 섹션에 raw 연도별 데이터 포함**
- 골든 형식:
  ```
  - 판매 시즌: 6월 초 ~ 9월 초 (3개월)
  - 피크 기간: 7월 중순~하순
  - 3개년 패턴: 일치 (±1주 이내)
  ```
- 초안 형식:
  ```
  - 2023년: 6월 초 시작 → 7월 중순 피크 → 8월 말 하락
  - 2024년: 6월 초 시작 → 7월 중순 피크 → 8월 말 하락
  - 2025년: 6월 초 시작 → 7월 초 피크 → 8월 중순 하락
  - 판매 시즌: ...
  - 피크 기간: ...
  - 3개년 패턴: ...
  ```
- 왜 문제: 골든 출력은 input의 raw 데이터를 그대로 노출하지 않고 요약만 제공. 초안은 input을 출력에 그대로 복사해 output 형식을 어김. 출력이 불필요하게 길어지고 가독성 저하.

**② 헤더 필드 순서 다름**
- 골든: 검색 기간 → 분석 날짜 → 데이터 출처 → 선정 기준
- 초안: 분석 날짜 → 데이터 출처 → 분석 대상 → 선정 기준
- 왜 문제: 필드 순서 불일치 + 골든에 없는 `분석 대상` 추가, 골든에 있는 `검색 기간` 누락.

**③ 진입 타이밍 섹션 표현 불일치**
- 골든: "현재 위치: 시즌 직전 (피크 2개월 전)" → 피크까지 거리를 병기
- 초안: "현재 위치: 시즌 2주 전" → 피크까지 거리 없음
- 왜 문제: 피크까지 거리가 진입 타이밍 판단에 중요한 정보인데 초안에서 빠짐.

## 3. 톤 차이 — 점수: 5/10 (보통)

말투·길이·격식 차이.

### 문제점

**① 권장 액션이 골든보다 장황함**
- 골든 패턴: 한 문장, 조건 → 결론 구조
  - `→ 공급처 확보 시 6월 초 진입 권장`
  - `→ 공급처 확보 시 즉시 진입 권장`
- 초안 패턴: 두 문장 이상, 비교 설명 포함
  - `→ 3개 키워드 중 시즌 진입이 가장 임박. 지금 바로 공급처 컨택 시작 권장.`
  - `→ 미니수박과 준비 일정 병행 가능. 6월 첫 주까지 공급처 확보 후 중순 진입 목표.`
- 왜 문제: 골든은 "공급처 확보 시" 같은 조건 문구로 사람 판단 여지를 명확히 남기지만, 초안은 타임라인까지 AI가 확정하는 어조.

**② 다른 품목과 비교하는 문장 포함**
- 예: `3개 키워드 중 시즌 진입이 가장 임박`, `미니수박과 준비 일정 병행 가능`
- 골든에는 개별 품목을 독립적으로 서술. 다른 품목과 교차 비교하는 문장 없음.
- 왜 문제: 리포트의 각 품목 섹션은 단독으로 읽혔을 때도 완결성 있어야 함. 다른 품목 언급은 가독성·확장성 저하.

## 4. 사실 오류/추측 — 점수: 6/10 (주의 필요)

입력 데이터에 없는 내용을 초안이 만들어낸 부분.

### 문제점

**① 미니수박 — "혹서기 품질 유지 확인"**
- 위치: 진입 판단 필요 사항 체크리스트
- 입력 데이터에 없는 내용. AI가 수박은 여름이라는 일반 상식으로 추가.
- 체크리스트는 사람이 확인해야 할 실제 항목이어야 하므로, 근거 없는 항목 추가는 혼선 유발.

**② 천도복숭아 — "산지 직거래 우선 검토"**
- 위치: 진입 판단 필요 사항 체크리스트
- 입력 데이터에 없는 내용. 공급 채널 전략을 AI가 임의로 지정.
- 직거래가 최선인지 여부는 사람이 판단해야 할 전략 결정.

**③ 피자두 — "일반 자두보다 유통망 좁음"**
- 위치: 진입 판단 필요 사항 체크리스트
- 입력 데이터에 없는 내용. AI가 일반적 지식으로 추론해 삽입.
- 유통망 현황은 실제 확인이 필요한 사항이므로, 사실처럼 서술하면 안 됨.

**④ 천도복숭아 — 판매 시즌 종료 시점 불일치**
- 초안: "6월 중순 ~ 9월 초 (약 2.5개월)"
- 골든(동일 품목): "6월 중순 ~ 8월 말 (2.5개월)"
- 3개년 중 2024년만 9월 초 하락. 나머지 2년은 8월 말. 입력 데이터의 최악값을 기준으로 잡은 것인지, 오류인지 불명확. 이런 경우 범위로 표기("8월 말~9월 초")하거나 골든 표현을 그대로 따르는 것이 안전.

## 5. AI↔사람 경계 위반 — 점수: 7/10 (명확한 위반)

사람이 봐야 할 결정을 AI가 처리한 부분.

### 문제점

**① 공급 전략 방향을 AI가 결정**
- 초안: "산지 직거래 우선 검토"
- 골든 패턴: "공급처 확보 가능 여부" → 확보 방법(직거래·중도매·위탁 등)은 체크리스트에 없음
- 왜 경계 위반: 어떤 채널로 공급처를 확보할지는 사람의 사업 전략 결정 영역. AI가 "우선"이라는 단어로 방향을 지정하면 안 됨.

**② 분석 범위 외 품목(신비복숭아) 언급**
- 초안 천도복숭아 권장 액션: "신비복숭아와 함께 묶음 전략 검토"
- 이번 입력 데이터에 신비복숭아 없음. AI가 golden/input-example.md의 다른 품목을 끌어다 전략을 만들어냄.
- 왜 경계 위반: 분석 범위를 AI가 임의로 확장. 사람이 허가하지 않은 데이터 참조.

**③ 구체적 타임라인을 AI가 확정**
- 초안: "6월 첫 주까지 공급처 확보 후 중순 진입 목표"
- 골든 패턴: "공급처 확보 시 6월 초 진입 권장" → 조건부, 확정 아님
- 왜 경계 위반: "6월 첫 주까지"라는 데드라인은 사람이 사업 상황 보고 결정해야 할 사항. AI가 확정형으로 서술하면 책임 소재 혼란 발생.
