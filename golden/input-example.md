# 인풋 예시 — 네이버 데이터랩 쇼핑인사이트 기반 농산물 진입 타이밍 분석

### 1단계: 네이버 데이터랩 쇼핑인사이트 검색
- 접속: 네이버 데이터랩 > 쇼핑인사이트
- 분야 설정: 식품 > 농산물 > 과일
- 기간 설정: 2025.05.22 ~ 2025.06.22 (오늘 26.05.22 기준 작년 동기 1개월)
- 결과: 인기검색어 1위~100위 확인

### 2단계: 세부 키워드 10개 선정
TOP 100 중에서 대형 키워드(사과, 포도, 복숭아 등)는 제외하고,  
세부 키워드만 선택:
- 사과 (X) → 청송사과 (O)
- 복숭아 (X) → 신비복숭아, 대극천복숭아 (O)
- 수박 (X) → 흑수박 (O)
- 자두 (X) → 피자두 (O)

선정된 10개 세부 키워드:
1. 청송사과
2. 신비복숭아
3. 흑수박
4. 피자두
5. 샤인머스캣
6. 미니수박
7. 천도복숭아
8. 대봉감
9. 거봉포도
10. 대극천복숭아

### 3단계: 각 키워드 검색어 트렌드 확인 (3개년 데이터)
네이버 데이터랩에서 각 키워드별 3년 데이터 조회:

**예시: 신비복숭아**
- 2023년: 6월 초 시작 → 7월 중순 피크 → 8월 말 하락
- 2024년: 6월 중순 시작 → 7월 하순 피크 → 9월 초 하락
- 2025년: 6월 초 시작 → 7월 중순 피크 → 8월 말 하락
- 현재 시점: 2026년 5월 22일
- 현재 위치: 시즌 직전 (피크 2개월 전)
- 판단: 시즌 지나지 않음 ✓, 검색량 증가 예상 ✓

**예시: 청송사과**
- 2023년: 9월 중순 시작 → 11월 피크 → 12월 말 하락
- 2024년: 9월 하순 시작 → 11월 중순 피크 → 12월 말 하락
- 2025년: 9월 중순 시작 → 11월 초 피크 → 12월 중순 하락
- 현재 시점: 2026년 5월 22일
- 현재 위치: 시즌 완전 종료 (비수기)
- 판단: 시즌 지남 X, 9월까지 대기 필요

**예시: 흑수박**

- 현재 위치: 시즌 초입

### 4단계: 데이터 기반 상품 선정 기준
각 키워드마다 2가지 확인:
1. 시즌이 지난 상품이 아닌가? (현재 5월 22일 기준)
2. 검색량이 증가할 상품인가? (3개년 패턴 일치 + 시즌 진입 시기)
